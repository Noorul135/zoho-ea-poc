# zoho-ea-poc
EA POC
Create a new empty GitHub repo, e.g. zoho-ea-poc.
From this folder:


bash   git init
   git add .
   git commit -m "Zoho EA POC"
   git branch -M main
   git remote add origin https://github.com/<you>/zoho-ea-poc.git
   git push -u origin main

.gitignore already excludes .env and node_modules/.
